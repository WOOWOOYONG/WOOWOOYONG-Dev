# House-Hotel 旅館預約服務

### 說明
- 火箭隊前端組協作作業（3人）

### 相關連結
- [Github Repo](https://github.com/TeamRocket12th/House-Hotel-Trio)
- [Demo](https://teamrocket12th.github.io/House-Hotel-Trio/#/)

### 功能

- 可以觀看 6 種房型的詳細資訊
- 可以用日曆方式，瀏覽未來 90 天已預約與尚未預約的時段
- 在選擇預約日期時，會即時顯示訂房價格總價
- 可以線上訂房，只能預約未來 90 天內的時段
- 若預約成功，會回傳成功通知
- 若預約失敗，會回傳訊息讓客戶知曉

### 練習內容

- Vue3+Vite
- Composition API
- Git Flow
