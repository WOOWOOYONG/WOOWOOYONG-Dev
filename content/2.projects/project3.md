# WOWOROOM床具電商平台

### 說明
- 火箭隊前端組協作作業（2人）


### 相關連結
- [Github Repo](https://github.com/WOOWOOYONG/WOWOROOM-OnlineShop)
- [Demo - 前台](https://tsaaiiiii.github.io/WOWOROOM-OnlineShop/)
- [Demo - 後台](https://tsaaiiiii.github.io/WOWOROOM-OnlineShop/admin)

### 功能
**前台使用者：**
- 可以選取產品加入購物車
- 可以送出訂購資料


**後台管理者：**
- 可以查看訂單明細
- 可以從圖表查看產品營收比重

### 練習內容
- 購物車CRUD
- Github Flow
- SASS BEM命名原則
- 使用axios串接API
- 使用C3.js圖表套件
